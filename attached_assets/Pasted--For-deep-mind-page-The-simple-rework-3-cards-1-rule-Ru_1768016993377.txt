If N is small, label it “early signal”.

1) Live Monitor tab → make it “Now”

NOW

2) Analysis & Patterns tab → make it “Your 3 loops”

Top 3 Triggers (last 30–90 days)

Top 3 Stabilizers

Example: sleep >6h, 2 meals, leaving the house

Top 3 Crash Loops

Work blowup → shutdown → insomnia → next-day low capacity

Urge spike → porn/sexting → shame → dissociation → insomnia

Each loop shows:

How often (count)

How recent

1 recommended interrupt (tiny)

No charts needed here—just clean bullets.

3) Visualizations tab → keep only 2 charts (the ones that matter)

Right now you have many charts that are easy to misread. Keep only:

Chart A: Sleep hours vs (Mood / Dissociation / Urges)

Show it as a simple bar/line with N and confidence.

Chart B: Driver frequency over time

A stacked bar: Sleep/Work/Loneliness/Pain/Urges per week.

This tells you “what’s been running my life lately.”

Remove (or demote) the “habit impact” and “synergy” charts until you have lots of data (N≥30–60 days). Otherwise it will confidently lie.
